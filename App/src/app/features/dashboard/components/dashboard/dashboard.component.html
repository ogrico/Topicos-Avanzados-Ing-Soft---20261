Bienvenido a la aplicación de Microservicios

Sobre esta aplicación

Esta es una aplicación Angular diseñada para practicar arquitectura de microservicios. Está estructurada para conectar con 4 microservicios serverless:

  • User Service: Gestión de usuarios - CRUD completo
  • Product Service: Gestión de productos del catálogo
  • Order Service: Gestión de pedidos y transacciones
  • Report Service: Generación de reportes y análisis

Configuración de Microservicios

Para conectar esta aplicación con tus microservicios serverless, edita los siguientes archivos:

1. Archivo de configuración: src/app/config/microservices.config.ts
2. Variables de entorno: src/environments/environment.ts (desarrollo) src/environments/environment.prod.ts (producción)

Nota: Actualmente, los servicios están configurados con URLs de ejemplo. Reemplázalas con las URLs reales de tus microservicios.
